## Provenance

Alumic is not a clean-room project. The design spec is grounded in two working prototypes that converge into the product ([ADR-054 Prototype convergence](/decisions/054-prototype-convergence)): **taleframe** (mature vector editor, substantial level editor, structural narrative graph editor, local DAM — SolidJS + Kobalte + Tailwind + Canvas2D + Three.js + Rust/WASM) and **geonode-engine** (82-node parametric graph with Blender-geometry-nodes parity, renderer-independent kernel, 358 passing tests). The spec describes the unified product; the Vision page names the prototypes explicitly so readers know which surfaces already ship versus which are net-new glue.
