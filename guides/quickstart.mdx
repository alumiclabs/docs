A two-minute path from nothing to a running template. Assumes Bun is installed.

<Steps>
  <Step title="Install the Alumic CLI">
    ```bash
    bun add -g @alumic/cli
    ```
  </Step>
  <Step title="Log in">
    ```bash
    alumic login
    ```
    Opens a device-flow login in your browser. The CLI caches a PAT under `~/.alumic/credentials` for subsequent commands. See [Authentication](/platform/authentication) and [ADR-026 Engine-plugin authentication](/decisions/026-engine-plugin-auth) for the credential model.
  </Step>
  <Step title="Create a project from a template">
    ```bash
    alumic create my-game --template topdown-action
    cd my-game
    ```
    Mints a server-issued project UUID ([ADR-025](/decisions/025-project-identity-server-uuid)), fetches the template bundle, and writes a local working directory stamped with `project.json`. The server is the source of truth; the local directory is a mirror for Git workflows and offline authoring.
  </Step>
  <Step title="Start the dev server">
    ```bash
    bun dev
    ```
    Runs Vite with SolidJS HMR for template-author iteration. See [CLI](/guides/cli) for the full command reference.
  </Step>
</Steps>
